# GSTFlow POC

## Project Overview

This project automates GST invoice processing using Optical Character Recognition (OCR).

The application extracts important invoice fields from PDF or image invoices and converts them into structured JSON. The extracted data can later be exported to CSV.

## Technologies

- Python
- OCR
- Tesseract OCR
- JSON
- CSV
- Git & GitHub
